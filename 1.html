HTML

Hypertext Markup Language (HTML) is the standard markup language for creating web pages and web application. Web browsers receive HTML documents from a web server or from lacal stotage and render them into multimedia web pages.

HTML describes the structure of a web pages semantically and originally includedcues for the appearance of the document.

HTML elements are the buliding blocks of HTML pages.

With HTML constructs, images and other objects, such as interactive forms, may be embedded into the rendered page.
